# docute-demo
基于 docute 实现的文档网站 demo，fork 即能使用，改改参数丰富代码即可

## 预览效果

* [docute-v3-demo](https://wugenqiang.github.io/docute-demo/docute-v3-demo/docs/#/)
* [docute-v4-demo](https://wugenqiang.github.io/docute-demo/docute-v4-demo/docs/#/)

## 本地部署

本地 Docute 服务开启

cd docs

npm i -g serve && serve .

或者使用 docute docs

> 以 docute-v3-demo 举例
